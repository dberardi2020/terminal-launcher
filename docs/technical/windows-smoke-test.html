7. /color paste key check

Launch with --inject-color (or injectColor true in settings). This is the one path not yet exercised live.

Failure mode (paste-specific): with paste it's all-or-nothing per pane. If nothing appears in a pane, Ctrl+V isn't reaching Claude's TUI (vs. the old typing path's mid-command corruption) — note which pane, and it's a one-function swap back if needed.
