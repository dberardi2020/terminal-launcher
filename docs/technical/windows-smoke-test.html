Windows smoke test

Verifying the windows-terminal-backend branch on Windows 11 — run this with your agent (Claude Code, on the PC). The agent drives the commands and verifies everything queryable — window geometry vs. the expected rects, window count, titles, the diag log, even the clipboard round-trip. You provide eyes on the few things no API exposes. Ticks + notes persist in your browser.

🤖 agent  runs a command / inspects window state and confirms 👁 you  needs your eyes (in-TUI or visual) 🤝 both  agent launches & measures, you glance to confirm

The agent can do the mechanical run end to end; realistically the only thing it can't check is whether /color actually tints the prompt bar inside Claude's TUI (§7) — there's no Windows API to read a wt pane's buffer, unlike iTerm2 on the Mac.

0. Setup

🤖 Agent assist: checks the branch and PATH, and can generate a scratch workspaces.json (a couple of panes + a single/split/quad workspace) so you don't have to hand-build one.
git checkout windows-terminal-backend
py -c "import shutil; print(shutil.which('wt'), shutil.which('claude'))"

1. Dry run

🤖 Agent assist: runs --dry-run and confirms the printed rects match _slot_rects(layout, flip) for your work area, plus the per-slot claude command + cwd. Fully automatable — no launch.
py -m terminal_launcher launch <ws> --dry-run

2. Single

🤖 Agent assist: after launch, enumerates CASCADIA_HOSTING_WINDOW_CLASS windows, reads the visible frame (DWMWA_EXTENDED_FRAME_BOUNDS), and confirms exactly one window filling the work area.
py -m terminal_launcher launch <single-ws>

3. Split

🤖 Agent assist: diffs both windows' visible frames against _slot_rects("split", flip), and — better than eyeballing — confirms the two frames abut with zero gap at the shared edge (the DWM-compensation check, numerically).

4. Combo optional

🤖 Agent assist: checks the three frames against the combo rects (one full-height left, two stacked right).

5. Quad with one empty slot

🤖 Agent assist: confirms exactly three windows (not four) at the correct quadrant rects — the empty quadrant having no window is the "real gap", verifiable by count + position.

6. Flip

🤖 Agent assist: checks the primary window's x is on the mirrored side vs. the unflipped rects.

7. /color paste your eyes

🤖 Agent assist: confirms the injection ran — no could not foreground / could not set clipboard warnings in the diag log — but cannot read Claude's TUI buffer on Windows, so whether the color actually landed is yours to see.

Launch with --inject-color (or injectColor true).

Failure mode (paste-specific): all-or-nothing per pane. If nothing appears, Ctrl+V isn't reaching the TUI (vs. the old typing path's mid-command corruption) — tell the agent which pane; it's a one-function swap back.

8. Clipboard restore

🤖 Agent assist: fully automatable — the agent sets a sentinel string on the clipboard before the launch, then reads it back afterward and confirms it's unchanged. No eyes needed.

9. Diagnostics

🤖 Agent assist: reads the log and reports any warnings verbatim.
py -m terminal_launcher logs      # or ~/.config/terminal-launcher/terminal-launcher.log
